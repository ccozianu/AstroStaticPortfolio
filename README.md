# StaticPortfolio

This is a static portfolio website built using HTML, CSS, and JavaScript. It features a responsive design and includes a contact form for visitors to get in touch.

## Features

- Responsive layout that works on all devices
- Modern and clean design
- Contact form with validation
- Smooth animations and transitions
- Easy to navigate and use

## Technologies Used

- HTML5
- CSS3
- JavaScript
- jQuery
- Bootstrap

## Installation

1. Clone the repository:
```
git clone https://github.com/username/static-portfolio.git
```

2. Open the `index.html` file in your web browser.

## Usage

1. Open the `index.html` file in your web browser.
2. Use the navigation menu to navigate to different sections of the portfolio.
3. Use the contact form to get in touch with the owner of the portfolio.
